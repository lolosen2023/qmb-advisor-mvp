# 住院垫付服务

##### 1.1住院垫付服务内容介绍

服务内容：客户**经二级及二级以上公立医院**确诊需要住院的，授权供应商协调安排网络城市**（覆盖范围详见附件一）**服务人员到医院，为客户提供不限次住院费用垫付，协助办理出院手续和理赔手续等服务。

|  |  |  |
|----|----|----|
| **序号** | **服务项目** | **项目说明** |
| **1** | 住院费用垫付 | 在获得保险公司授权确认后，**授权供应商在其保险责任保额范围内**为客户垫付因疾病或意外住院产生的医疗费用。**垫付的医疗费用在医疗责任保险金中扣除** |

##### 1.2住院垫付服务流程说明

流程说明：

1)  拨打服务热线028-85359098申请服务/巴杏健康公众号/小程序，在“我的服务 -住院垫付服务”，点击进入服务页面进行申请；

2)  健康专员与客户联系，收集客户资料（收集住院通知书、就诊病历等相关资料），审核通过后进行入院指导；

3)  审核客户材料，进行资料审核、理赔初审、垫付审核；

4)  与客户签署住院垫付协议，进行住院费用垫付，监控治疗费用，视情况追加垫付费用；

5)  协助客户进行出院结算，收集出院小结等客户资料，陪同出院；

6)  协助理赔，理赔结案。

##### 1.3**使用资格：**

如下情形不符合住院费用垫付服务使用资格

1）存在既往病史但在投保时未如实告知保险公司；

2）未达到保险条款约定的免赔额；

3）本保单年度保额已赔满；

4）保单等待期内申请服务；

5）住院事由属于保单责任免除事项；

6）医院等级或所在位置不符合服务范围清单；

7）单次住院垫付金额达到或超过协议约定垫付限额；

8）出院时因需要社会基本医疗保险、城乡居民大病保险报销或其他原因等，导致被保险人不能在出院时向乙方提供医疗费用发票原件用于保险理赔。
