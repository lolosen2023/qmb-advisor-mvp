# 全民保尊享版智能客服销售助手知识库

本文件夹用于沉淀“全民保尊享版”客服问答与销售智能助手的统一知识底稿。

## 文件

- `全民保尊享版_客服销售助手知识库.md`：整合后的主文档，包含产品事实、客服问答、销售辅助话术、合规红线、理赔和服务流程。

## 使用原则

1. 优先使用“全民保尊享版 - 更新”目录中的新版资料。
2. 旧问答、口播卡仅作为表达方式参考；如与新版方案冲突，以新版方案为准。
3. 销售和客服输出不得承诺“什么都赔”“花多少报多少”“终身保障”“免费保险”“无社保也能赔”等高风险说法。
4. 涉及理赔结论、医院/药店/药品清单实时状态、退保金额、保单有效性等问题，应引导客户以保单、保险公司审核或官方客服为准。
